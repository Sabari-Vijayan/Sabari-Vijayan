### 🚀 Featured Projects

#### **Infrastructure & Tooling**
* **[Tinkerfetch](https://github.com/Sabari-Vijayan/tinkerfetch.git)**: A custom system info tool (Fastfetch skin) with a dedicated mode for tracking community events.
* **[Gemini-CLI-Skills](https://github.com/Sabari-Vijayan/Gemini-CLI-Skills.git)**: A specialized repository of `.md` skill definitions to enhance Gemini CLI workflows.
* **[KTU Status Tracker](https://status.app.ktu.in)**: Uptime monitor for the KTU website built using Upptime and GitHub Actions.

#### **Full-Stack Applications**
* **[SmartPlace](https://github.com/Sabari-Vijayan/SmartPlace)**: A self-hostable placement management system for colleges. Built with **React/Vite, Express, Supabase**, and **Redis** for caching.
* **[Code-A-Pookalam 2025](https://github.com/Sabari-Vijayan/CODE-A-POOKALAM-2025-COMPETITION.git)**: Official website for the TinkerHub RIT competition.
* **[Christmas Secret Messages](https://github.com/Sabari-Vijayan/christmas.git)**: An anonymous messaging platform where gifts (messages) are unlocked strictly on Christmas Day. (Supabase/React).
* **[CarTinder](https://github.com/Sabari-Vijayan/cartinder.git)**: A MERN stack application applying the "swipe" discovery mechanic to car listings.

#### **CMS & UI**
* **[TinkerHub Ghost](https://github.com/Sabari-Vijayan/tinkerhub-ghost.git)**: Custom Ghost CMS theme developed for TinkerHub RIT during the Ghosted Hackathon.
